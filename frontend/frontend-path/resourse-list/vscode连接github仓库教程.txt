步骤一 建库：
    1.登录你的GitHub。
    2.点击右上角 + -> New repository。
    3.填写新仓库的名称（如 my-new-project），添加描述（可选）。
    4.非常重要：不要勾选 Initialize this repository with a README。因为我们是要推送已有的代码，而不是从GitHub克隆一个空的下来。
    5.点击 Create repository 创建。
    6.仓库的URL（类似 https://github.com/your-username/my-new-project.git）

步骤二 配置本地文件如dailyLife并连接github：
    1.在VS Code中按 Ctrl+` （反引号键）打开集成终端。
    2. 本地配置，并连接远程仓库：
        echo "# frontend" > README.md

        # 3. 初始化Git
        git init

        # 4. 添加文件到暂存区
        git add README.md

        # 5. 提交（此时应该能成功）
        git commit -m "first commit"      //提交文件到git

        # 后续操作
        git branch -M main
        git remote add origin https://github.com/kale987654/frontend.git          //git用来远程连接github/仓库的，一次就行
        git push -u origin main         //将提交文件推送到仓库

可能用到的指令：
    1.在终端里输入以下命令，查看当前本地项目关联的远程仓库是什么：
        bash
        git remote -v

    2.本地更换或添加新仓库：
        情况A：如果已有 origin（需要更换）：
        使用 set-url 命令来修改它。
            bash
            git remote set-url origin https://github.com/your-username/my-new-project.git
            请将上面的URL替换成你在第一步创建的新仓库的URL！

        情况B：如果没有 origin（全新的本地项目）：
            使用 add 命令来添加一个新的远程仓库。
            bash
            git remote add origin https://github.com/your-username/my-new-project.git

    3.在VS Code的终端中，运行推送命令：
        bash
        git push -u origin main
        这是向github仓库上传本地变更记录，并更新仓库。
